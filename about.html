ABOUT 2KIO

Description

>  2kio Events is a website made in Kenya by kenyan students.It is a web page where users can create and/or choose events to join according to their preferences.

>  The 2Kio web page users can choose an event based on their prefereces.There are select categories which make it easier for users to identify events they're interested in.The various events include:

  • Education
  • Art
  • Technology
  • Health
  • Religion
  • Sports

    • >  Users can also join events that will be shown under the name of events under each and every category.

    Technologies Used

    >The 2Kio webpage was created using three Programming languages;

  • HyperText Markup Language(HTML)
  • Cascading StyleSheet(CSS)
  • JavaScript and jQuery
